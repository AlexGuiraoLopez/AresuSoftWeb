Cep Fly

Cep Fly es un prototipo de mi primera aplicación para dispositivos móviles Android escrita con Java y Kotlin.

La aplicación permite realizar distintas funciones según el tipo de usuario que la utiliza:

Administrador: Dar de alta a trabajadores, asignarlos al vuelo y visualizar su información. Necesita ser aprobado mediante login.

Cliente: Puede comprar billetes y visualizar la información del vuelo.

Es extensión de un proyecto anterior realizado en java sin interfaz gráfica. He aprendido conceptos básicos sobre la creación de layouts y activities.
