Cep Fly

Cep Fly es un prototipo de mi primera aplicación para dispositivos móviles Android escrita con Java y Kotlin.

La aplicación permite realizar distintas funciones según el tipo de usuario que la utiliza (administrador o cliente).

Administrador: Puede dar de alta a trabajadores para asignarlos al vuelo y visualizar la información de dichos trabajadores. Necesita ser aprobado mediante un nombre de usuario y contraseña.

Cliente: Puede comprar billetes y visualizar la información del vuelo.

La aplicación es una extensión de un proyecto anterior realizado en java pero sin interfaz gráfica. Con esta implementación he podido aprender conceptos básicos sobre la creación de layouts y activities con Android Studio así como empezar a programar con Kotlin.
